# 二、产品说明
## 2.1、产品背景
### 2.1.1、产品存在的意义
- 需要个人站点来增加个人技术影响力
- 需要将日常的技术应用于个人项目，加深对技术的理解

### 2.1.2、用户定位
暂定为于个人站点，后续有机会可以拓展成多用户体系

### 2.1.3、产品定位
- 需要支持博客的发布、管理、统计等日常功能
